# Tarea-02
Sitio donde se aloja los datos correspondientes a la tarea 02 del curso de programación SIG.
Este repositorio contiene el notebook y los archivos CSV utilizados para el análisis de una base de datos de activos de un acueducto municipal.

Se incluye:
- carga de datos con pandas
- limpieza básica de datos
- filtrado y agrupamiento
- estadísticas descriptivas
- gráficos con matplotlib
